The Jelly Button

Newton's laws applied to UI. Action has reaction. When you click, the button doesn't just change color; it absorbs the energy of the press, squashing and wobbling to show weight and mass.

HTML Structure

<button class="btn-jelly" id="jellyBtn">Squish Me</button>

CSS Base Styles

.btn-jelly {
    background: var(--accent-fluid);  /* #10b981 */
    color: #000;
    padding: 1.2rem 3.5rem;
    border-radius: 50px;
    border: none;
    font-weight: 700;
    font-size: 1.1rem;
    cursor: pointer;
    /* GSAP handles scaleX/scaleY animation */
}

GSAP Animation

Timeline Sequence

const jellyBtn = document.getElementById('jellyBtn');

jellyBtn.addEventListener('click', () => {
    gsap.timeline()
        // Stage 1: SQUASH (press energy absorbed)
        .to(jellyBtn, {
            scaleX: 1.25,    // Wider
            scaleY: 0.75,    // Shorter
            duration: 0.1
        })
        // Stage 2: STRETCH (recoil)
        .to(jellyBtn, {
            scaleX: 0.75,    // Narrower
            scaleY: 1.25,    // Taller
            duration: 0.1
        })
        // Stage 3: SETTLE (elastic bounce back)
        .to(jellyBtn, {
            scaleX: 1,
            scaleY: 1,
            duration: 0.8,
            ease: "elastic.out(1, 0.3)"
        });
});

Animation Breakdown

/* GSAP Timeline creates a sequence of animations.

   Stage 1 (0.1s): Squash
   - scaleX increases (button gets wider)
   - scaleY decreases (button gets shorter)
   - Simulates compression from the click force
   
   Stage 2 (0.1s): Stretch  
   - Inverse of squash (narrow + tall)
   - Simulates elastic recoil
   
   Stage 3 (0.8s): Settle
   - Returns to original scale
   - elastic.out(1, 0.3) creates the wobble effect
     - 1 = amplitude (how far it overshoots)
     - 0.3 = period (wobble frequency)
*/
