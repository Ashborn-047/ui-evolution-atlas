Glass Slider

The slider thumb casts a soft shadow, simulating a physical knob on frosted glass. Frosted friction creates a premium feel.

CSS

.glass-slider-track {
    width: 250px;
    height: 6px;
    background: rgba(255,255,255,0.1);
    border-radius: 10px;
    position: relative;
}

.glass-slider-fill {
    height: 100%;
    width: 50%;
    background: var(--accent-glass);
    border-radius: 10px;
    box-shadow: 0 0 10px var(--accent-glass);
}

.glass-slider-thumb {
    position: absolute;
    left: 50%;
    top: 50%;
    transform: translate(-50%, -50%);
    width: 24px;
    height: 24px;
    background: white;
    border-radius: 50%;
    cursor: grab;
    box-shadow: 0 0 15px rgba(255,255,255,0.5);
}

GSAP Draggable

Draggable.create("#glassThumb", {
    type: "x",
    bounds: ".glass-slider-track",
    onDrag: function() {
        const progress = (this.x + 125) / 250;  // 0 to 1
        gsap.set("#glassFill", { width: progress * 100 + "%" });
    }
});
