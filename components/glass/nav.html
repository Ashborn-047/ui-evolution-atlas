Dock Navigation

Navigation creates a "fisheye" lens effect expanding the hit area of active elements. Scale and translateY create the magnification illusion.

CSS

.dock-container {
    display: flex;
    gap: 15px;
    background: rgba(255,255,255,0.05);
    padding: 15px 25px;
    border-radius: 24px;
    backdrop-filter: blur(12px);
    border: 1px solid rgba(255,255,255,0.1);
}

.dock-item {
    width: 45px;
    height: 45px;
    background: rgba(255,255,255,0.1);
    border-radius: 14px;
    display: flex;
    align-items: center;
    justify-content: center;
    cursor: pointer;
    transition: 0.2s cubic-bezier(0.25, 0.46, 0.45, 0.94);
}

.dock-item:hover {
    transform: scale(1.3) translateY(-10px);
    background: var(--accent-glass);
    box-shadow: 0 10px 20px rgba(0,0,0,0.5);
}
