Stacked Cards

The carousel evolved into a deck. Hover to expand and see all cards using blur and scale on background items.

Message 1
Message 2
Message 3

CSS

.stack-wrapper {
    position: relative;
    width: 300px;
    height: 160px;
}

.stack-card {
    position: absolute;
    width: 100%;
    height: 70px;
    background: rgba(255,255,255,0.08);
    backdrop-filter: blur(10px);
    border-radius: 16px;
    transition: 0.4s cubic-bezier(0.34, 1.56, 0.64, 1);
}

/* Stacked state */
.stack-card:nth-child(1) { transform: translateY(0) scale(1); z-index: 3; }
.stack-card:nth-child(2) { transform: translateY(15px) scale(0.92); opacity: 0.6; }
.stack-card:nth-child(3) { transform: translateY(30px) scale(0.84); opacity: 0.3; }

/* Expanded on hover */
.stack-wrapper:hover .stack-card:nth-child(2) { 
    transform: translateY(80px) scale(1); 
    opacity: 1; 
}
.stack-wrapper:hover .stack-card:nth-child(3) { 
    transform: translateY(160px) scale(1); 
    opacity: 1; 
}
