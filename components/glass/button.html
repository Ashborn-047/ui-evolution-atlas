Prism Button

Instead of heavy shadows, we use light. The Prism button catches a gradient shine sweep on hover.

CSS

.btn-prism {
    position: relative;
    background: rgba(255,255,255,0.05);
    border: 1px solid rgba(255,255,255,0.2);
    padding: 15px 40px;
    border-radius: 50px;
    color: white;
    overflow: hidden;
    backdrop-filter: blur(5px);
}

/* The light sweep effect */
.btn-prism::before {
    content: '';
    position: absolute;
    top: 0;
    left: -100%;
    width: 100%;
    height: 100%;
    background: linear-gradient(
        90deg, 
        transparent, 
        rgba(255,255,255,0.4), 
        transparent
    );
    transition: 0.5s;
}

.btn-prism:hover::before {
    left: 100%;  /* Sweeps across */
}
