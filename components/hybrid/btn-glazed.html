Glazed Jelly

The perfect hybrid: combines the glossy gradient shine of the Glass Age with the satisfying squish physics of the Organic Flow era. Click it to feel the jelly!

HTML Structure

<button class="btn-glazed-jelly" id="glazedBtn">
    Squish + Shine
</button>

CSS: Glass + Gradient

.btn-glazed-jelly {
    /* Glass Era: Gradient background */
    background: linear-gradient(
        135deg, 
        rgba(139, 92, 246, 0.8),   /* Purple */
        rgba(6, 182, 212, 0.8)     /* Cyan */
    );
    
    padding: 16px 40px;
    border-radius: 50px;
    
    /* Glass Era: Semi-transparent border */
    border: 1px solid rgba(255,255,255,0.4);
    
    color: white;
    font-weight: 800;
    cursor: pointer;
    
    /* Glass Era: Layered shadows for depth */
    box-shadow: 
        0 10px 20px rgba(139, 92, 246, 0.3),  /* Outer glow */
        inset 0 2px 0 rgba(255,255,255,0.4);   /* Top shine */
    
    /* Glass Era: Blur behind */
    backdrop-filter: blur(5px);
    
    /* Fluid Era: Transform origin for squish */
    transform-origin: center center;
}

GSAP: Jelly Squish Animation

const glazedBtn = document.getElementById('glazedBtn');

glazedBtn.addEventListener('click', () => {
    gsap.timeline()
        // Stage 1: SQUASH (compress on impact)
        .to(glazedBtn, { 
            scaleX: 1.2,      // Wider
            scaleY: 0.8,      // Shorter
            duration: 0.1 
        })
        // Stage 2: STRETCH (elastic recoil)
        .to(glazedBtn, { 
            scaleX: 0.9,      // Narrower
            scaleY: 1.1,      // Taller
            duration: 0.1 
        })
        // Stage 3: SETTLE (wobble back)
        .to(glazedBtn, { 
            scaleX: 1, 
            scaleY: 1, 
            duration: 0.8,
            ease: "elastic.out(1, 0.3)"
        });
});

/* elastic.out(amplitude, period):
   - amplitude: 1 = normal overshoot
   - period: 0.3 = faster oscillation (more wobbles)
   
   Lower period = more jelly-like wobble */

The Hybrid Philosophy

/* This button combines two eras:

   GLASS AGE Techniques:
   ├── linear-gradient background
   ├── Semi-transparent border
   ├── Layered box-shadows (outer + inset)
   └── backdrop-filter: blur()
   
   FLUID ERA Techniques:
   ├── GSAP timeline animation
   ├── scaleX/scaleY deformation
   └── elastic.out easing
   
   Result: A button that LOOKS premium (glass)
   and FEELS alive (fluid). */
