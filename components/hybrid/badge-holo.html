Holographic Badge

Solid base with a sweeping light effect on hover, like a holographic sticker.

HOLOGRAPHIC

CSS

.badge-holo {
    background: linear-gradient(135deg, #222, #333);
    padding: 8px 16px;
    border: 1px solid #444;
    position: relative;
    overflow: hidden;
}

/* Light sweep on hover */
.badge-holo::before {
    content: '';
    position: absolute;
    top: -50%; left: -50%;
    width: 200%; height: 200%;
    background: linear-gradient(
        45deg, 
        transparent, 
        rgba(6, 182, 212, 0.3), 
        transparent
    );
    transform: rotate(45deg) translateY(100%);
    transition: transform 0.5s;
}

.badge-holo:hover::before {
    transform: rotate(45deg) translateY(-100%);
}

.badge-holo:hover {
    border-color: var(--accent-cyan);
}
