Rotary Knob

A physical dial control brought to the digital screen. Uses GSAP Draggable with type:"rotation" for smooth, bounded circular motion with inertia.

Value: 0°

HTML Structure

<div class="rotary-container">
    <div class="rotary-knob" id="rotaryKnob">
        <div class="rotary-marker"></div>
    </div>
</div>

CSS: Container and Knob

.rotary-container {
    width: 80px;
    height: 80px;
    background: #111;
    border-radius: 50%;
    border: 2px solid #333;

.rotary-knob {
    width: 100%;

GSAP Draggable: Rotation Type

Draggable.create("#rotaryKnob", {
    type: "rotation",  // Rotate instead of x/y
    
    bounds: {
        minRotation: -135,
        maxRotation: 135
    },
    
    inertia: true,  // Momentum after release
    
    onDrag: function() {
        // Get current rotation value
        const rotation = Math.round(this.rotation);
        document.getElementById('knobValue').textContent = rotation;
    },
    
    onThrowUpdate: function() {
        // Also update during inertia
        const rotation = Math.round(this.rotation);
        document.getElementById('knobValue').textContent = rotation;
    }
});

/* type: "rotation" key differences:
   
   1. Drag direction = rotation (circular motion)
   2. this.rotation gives current angle
   3. bounds.minRotation / maxRotation limit range
   4. inertia: true adds momentum spin */

Use Cases

/* Rotary knobs are perfect for:
   
   - Volume controls
   - EQ frequency adjustments
   - Thermostat temperature
   - Precision value tuning
   - Simulating audio/video equipment
   
   The bounded rotation (-135° to +135°) matches
   real physical knobs that have start/end stops. */
