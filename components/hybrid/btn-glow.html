Atmospheric Glow

Combines a solid button structure with a fluid internal glow that tracks cursor position. Uses CSS custom properties updated via JavaScript to position a radial gradient.

HTML Structure

<button class="btn-hybrid" id="hybridBtn">
    <span>Initialize</span>
</button>

CSS: Custom Property Glow

.btn-hybrid {
    position: relative;
    padding: 16px 40px;
    background: #050505;
    color: white;
    border: 1px solid #333;
    border-radius: 12px;
    overflow: hidden;
}

/* The glow layer uses CSS custom properties */
.btn-hybrid::before {
    content: '';
    position: absolute;
    inset: 0;
    
    /* Radial gradient positioned at --x, --y */
    background: radial-gradient(
        circle at var(--x, 50%) var(--y, 50%),
        rgba(139, 92, 246, 0.3),
        transparent 70%
    );
    
    opacity: 0;
    transition: opacity 0.3s;
    z-index: 1;
}

/* Text stays above glow */
.btn-hybrid span {
    position: relative;
    z-index: 2;
}

.btn-hybrid:hover::before {
    opacity: 1;
}

.btn-hybrid:hover {
    border-color: var(--accent-hybrid);
    box-shadow: 0 0 30px -5px rgba(139, 92, 246, 0.4);
}

JavaScript: Update Custom Properties

const hybridBtn = document.getElementById('hybridBtn');

hybridBtn.addEventListener('mousemove', (e) => {
    const rect = hybridBtn.getBoundingClientRect();
    
    // Calculate mouse position relative to button
    const x = e.clientX - rect.left;
    const y = e.clientY - rect.top;
    
    // Update CSS custom properties
    hybridBtn.style.setProperty('--x', `${x}px`);
    hybridBtn.style.setProperty('--y', `${y}px`);
});

/* The CSS radial-gradient reads --x and --y,
   so the glow center follows the cursor in real-time. */

Why CSS Custom Properties?

/* Benefits of this approach:

   1. No GSAP required - just vanilla JS + CSS
   2. Hardware accelerated - gradient is GPU-rendered
   3. Smooth - no layout thrashing
   4. Default fallback - var(--x, 50%) centers if unset
   
   The technique: JS updates the property, CSS applies it
   instantly to the gradient position. Much smoother than
   manipulating element positions directly. */
