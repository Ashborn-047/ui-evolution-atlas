Liquid Slider

A slider thumb that deforms like a liquid droplet when dragged. Combines a solid track with fluid physics using GSAP Draggable's border-radius morphing.

HTML Structure

<div class="liquid-slider-track">
    <div class="liquid-slider-thumb" id="liquidThumb"></div>
</div>

CSS: Track and Thumb

.liquid-slider-track {
    width: 250px;
    height: 10px;
    background: #333;
    border-radius: 5px;
    position: relative;
    box-shadow: inset 0 2px 5px rgba(0,0,0,0.5);
}

.liquid-slider-thumb {
    width: 30px;
    height: 30px;
    background: var(--accent-hybrid);  /* Purple */
    border-radius: 50%;  /* Circle by default */
    position: absolute;
    top: 50%; left: 0;
    transform: translate(-50%, -50%);
    cursor: grab;
    box-shadow: 0 0 15px var(--accent-hybrid);
    
    /* Smooth radius transitions */
    transition: border-radius 0.2s;
}

GSAP Draggable with Deformation

Draggable.create("#liquidThumb", {
    type: "x",
    bounds: ".liquid-slider-track",
    
    onDragStart: function() {
        // Deform to droplet shape
        gsap.to(this.target, {
            borderRadius: "40% 60% 60% 40% / 40% 50% 50% 60%",
            scale: 1.2,
            duration: 0.2
        });
    },
    
    onDragEnd: function() {
        // Return to circle
        gsap.to(this.target, {
            borderRadius: "50%",
            scale: 1,
            duration: 0.4,
            ease: "elastic.out(1, 0.5)"
        });
    }
});

Border-Radius Deformation

/* Complex border-radius syntax:
   
   border-radius: TL TR BR BL / TL TR BR BL;
                  ──────────────  ──────────────
                  Horizontal      Vertical
                  radii           radii
   
   "40% 60% 60% 40% / 40% 50% 50% 60%"
   
   This creates an asymmetric blob shape that looks
   like a water droplet being dragged.
   
   When released, elastic.out bounces it back to 50%
   (a perfect circle) with a satisfying wobble. */
