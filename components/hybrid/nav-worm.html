Worm Line Nav

An animated underline that crawls like a worm between navigation items. Combines a solid underline structure with fluid animation using GSAP to animate width and position.

<div class="nav-worm" id="wormNav">
    <div class="worm-line" id="wormLine"></div>

JavaScript: The Worm Animation

function moveWorm(el) {
    const line = document.getElementById('wormLine');
    const wrapperRect = el.parentElement.getBoundingClientRect();
    const itemRect = el.getBoundingClientRect();
    
    // Calculate position relative to parent
    const newLeft = itemRect.left - wrapperRect.left;
    const newWidth = itemRect.width;
    
    gsap.to(line, {
        left: newLeft,
        width: newWidth,
        duration: 0.4,
        ease: "expo.out"  // Fast start, slow end
    });
}

/* expo.out easing makes the line "shoot" to the new
   position quickly, then ease into place - like a
   worm stretching and snapping to its target. */

Animation Breakdown

/* The worm effect comes from animating BOTH:

   1. left position - where the line starts
   2. width - how long the line is
   
   When moving RIGHT:
   - Line stretches (width grows)
   - Then snaps to new position (left changes)
   
   When moving LEFT:
   - Same but reversed
   
   The expo.out easing (fast→slow) creates the
   organic "crawl" feel. Linear easing would
   feel more mechanical. */
