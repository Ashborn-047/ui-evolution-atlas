Neon Depth Card

Combines a solid dark container with a continuously rotating conic gradient that creates a neon border effect on hover. The rotating light simulates energy flowing through the card.

Neon Breath

HTML Structure

<div class="card-neon-depth">
    <h3>Neon Breath</h3>
</div>

CSS Technique: Rotating Border

.card-neon-depth {
    width: 300px;
    height: 180px;
    background: #050505;
    border-radius: 20px;
    position: relative;
    overflow: hidden;
}

/* The rotating gradient - hidden by default */
.card-neon-depth::before {
    content: '';
    position: absolute;
    top: -50%; left: -50%;
    width: 200%; height: 200%;
    
    /* Conic gradient creates the sweep effect */
    background: conic-gradient(
        transparent, 
        var(--accent-cyan),  /* The neon color */
        transparent 30%
    );
    
    /* Continuous rotation */
    animation: rotate 4s linear infinite;
    
    /* Hidden until hover */
    opacity: 0;
    transition: opacity 0.3s;
}

/* Show on hover */
.card-neon-depth:hover::before {
    opacity: 1;
}

/* Inner card covers the center, leaving only border visible */
.card-neon-depth::after {
    content: '';
    position: absolute;
    inset: 2px;  /* 2px gap = border width */
    background: #111;
    border-radius: 18px;
    z-index: 1;
}

Animation Keyframe

@keyframes rotate {
    0% {
        transform: rotate(0deg);
    }
    100% {
        transform: rotate(1turn);  /* Same as 360deg */
    }
}

/* The ::before pseudo-element is 200% the size of the card
   and rotates around its center, creating the sweep effect.
   
   The ::after element covers everything except the 2px edge,
   so only the rotating gradient edge is visible as a border. */

Why This Works

/* Layering order:
   
   1. Base card (.card-neon-depth) - dark background
   2. ::before - Large rotating conic gradient (z-index: 0)
   3. ::after - Inner cover that hides center (z-index: 1)
   4. Content (h3) - Uses position: relative (z-index: 2)
   
   The conic-gradient only has color in ~30% of its arc,
   so as it rotates, it looks like a light beam sweeping
   around the border.
   
   No JavaScript required - pure CSS animation! */
