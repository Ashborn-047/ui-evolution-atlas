Morphing Field

Solid underline input with Fluid floating label. The label shrinks and floats up on focus.

CSS

.hybrid-input-group {
    position: relative;
}

.hybrid-input {
    width: 100%;
    padding: 15px;
    background: transparent;
    border: none;
    border-bottom: 2px solid #333;  /* Solid underline */
    color: white;
    outline: none;
}

.hybrid-input:focus {
    border-color: var(--accent-hybrid);
}

.hybrid-label {
    position: absolute;
    left: 15px; top: 15px;
    color: #666;
    pointer-events: none;
    transition: 0.3s;  /* Fluid morph */
}

/* Float up on focus or when filled */
.hybrid-input:focus ~ .hybrid-label,
.hybrid-input:not(:placeholder-shown) ~ .hybrid-label {
    top: -10px;
    left: 0;
    font-size: 0.75rem;
    color: var(--accent-hybrid);
}
