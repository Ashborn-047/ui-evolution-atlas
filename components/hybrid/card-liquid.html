Liquid Solid Card

Solid rigid container morphs into organic shape. A fluid blob rises from below on hover.

Solid + Morph

CSS

.card-liquid-solid {
    background: #111;
    border: 2px solid #333;
    border-radius: 0;  /* Starts rigid */
    overflow: hidden;
    position: relative;
}

/* Rising blob */
.card-liquid-solid::before {
    content: '';
    position: absolute;
    width: 300px;
    height: 300px;
    background: var(--accent-hybrid);
    border-radius: 50%;
    top: 100%; left: 50%;
    transform: translateX(-50%);
    transition: top 0.6s cubic-bezier(0.68, -0.55, 0.265, 1.55);
}

.card-liquid-solid:hover {
    /* Solid → Organic shape morph */
    border-radius: 50px 20px 50px 20px;
    border-color: transparent;
}

.card-liquid-solid:hover::before {
    top: -20px;  /* Blob rises up */
}
