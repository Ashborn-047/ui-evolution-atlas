Fluid Stream

Glass cards with 3D perspective tilt. Hover on one to push siblings.

CSS

.hybrid-stream {
    display: flex;
    gap: 15px;
    perspective: 1000px;
}

.stream-card {
    width: 100px;
    height: 140px;
    background: linear-gradient(180deg, rgba(30,30,30,0.8), rgba(10,10,10,0.9));
    border: 1px solid #333;
    border-radius: 16px;
    transition: 0.4s;
}

.stream-card:hover {
    transform: translateY(-20px) rotateX(10deg);
    border-color: var(--accent-hybrid);
    box-shadow: 0 20px 30px -10px rgba(0,0,0,0.5);
}

/* Push siblings */
.stream-card:hover ~ .stream-card {
    transform: translateX(10px);
}
