Spatial Glass Card

Combines Glass (blur, translucency) with Fluid (glowing blob background). The blob moves on hover.

SPATIAL

Glass + Glow

Structure meets fluid light.

CSS - Glass Layer

.card-hybrid {
    background: rgba(20, 20, 25, 0.4);
    backdrop-filter: blur(20px);
    border: 1px solid rgba(255, 255, 255, 0.08);
    border-radius: 24px;
}

.card-hybrid:hover {
    transform: translateY(-5px) scale(1.02);
    box-shadow: 0 20px 40px rgba(0,0,0,0.5), 
                inset 0 0 20px rgba(139, 92, 246, 0.1);
}

CSS - Fluid Blob

.hybrid-blob-bg {
    position: absolute;
    width: 100px;
    height: 100px;
    background: var(--accent-hybrid);
    filter: blur(50px);
    opacity: 0.4;
    transition: 0.5s;
}

.card-hybrid:hover ~ .hybrid-blob-bg {
    transform: translate(50px, -20px);
    opacity: 0.6;
}
