The Card

In the early web, "depth" was a binary illusion. Cards didn't float; they physically shifted position on X/Y axes to mimic a depressed button. This hard-edge feedback confirmed interaction reliability on slow connections.

File Record

Content contained in rigid boxes. Hover triggers a mechanical shift and negative filter.

HTML Structure

<div class="card-solid">
    <h3>File Record</h3>
    <p>Card description text.</p>
</div>

CSS Techniques

Key Properties

.card-solid {
    /* Base styling */
    background: #1a1a1a;
    border: 2px solid #333;
    padding: 2rem;
    
    /* Stepped transition for mechanical feel */
    transition: all 0.1s steps(2);
    cursor: crosshair;
}

.card-solid:hover {
    /* Invert colors */
    background: #ccc;
    border-color: #fff;
    
    /* Mechanical displacement */
    transform: translate(-4px, -4px);
    
    /* Hard shadow offset */
    box-shadow: 8px 8px 0 var(--accent-solid);
}

.card-solid:active {
    /* Press-in state */
    transform: translate(0, 0);
    box-shadow: none;
}

Animation Notes

/* The key technique here is `steps(2)` in the transition.
   
   Instead of smooth interpolation, this creates a 
   robotic, mechanical feel by jumping between states.
   
   Combined with the hard box-shadow offset, it mimics
   physical button behavior from early UI systems. */
