Mechanical Toggle

Toggles weren't smooth sliders; they were switches borrowed from hardware electronics. Instant state change.

POWER

CSS

.mech-switch {
    width: 50px;
    height: 25px;
    background: #333;
    position: relative;
    cursor: pointer;
}

.mech-switch::after {
    content: '';
    position: absolute;
    left: 2px;
    top: 2px;
    width: 21px;
    height: 21px;
    background: #888;
    transition: left 0s;  /* Instant - no animation! */
}

.mech-switch.active {
    background: var(--accent-solid);
}

.mech-switch.active::after {
    left: 27px;
    background: white;
}

JavaScript

// Simple toggle
element.onclick = () => element.classList.toggle('active');
