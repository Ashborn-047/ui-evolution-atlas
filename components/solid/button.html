Chiseled Button

Buttons designed to look heavy. The chiseled shadow effect tells users: "This is clickable." Press to see the tactile feedback.

CSS Technique

.btn-chiseled {
    background: var(--accent-solid);
    color: white;
    font-family: var(--font-mono);
    font-weight: bold;
    border: none;
    padding: 15px 30px;
    
    /* The "depth" effect - offset shadow */
    box-shadow: 6px 6px 0px #1e40af;
    
    cursor: pointer;
    transition: transform 0.1s, box-shadow 0.1s;
}

/* Press-in state */
.btn-chiseled:active {
    transform: translate(3px, 3px);
    box-shadow: 3px 3px 0px #1e40af;
}
